# Worker Info Retrieval

## Description
This stack retrieves worker information based on the provided run ID or an override run ID.

## Variables

### Required
None

### Optional
| Name | Description | Default |
|------|-------------|---------|
| overide_run_id | Alternative run ID for worker lookup | "null" |
